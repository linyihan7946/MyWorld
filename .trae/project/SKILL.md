# Project Skills & Architecture

## Tech Stack
- **Language**: TypeScript
- **Framework**: Vue 3
- **Build Tool**: Vite
- **3D Engine**: Three.js

## Architecture
- **Core**: Generic 3D engine components (Scene, Renderer, Loop).
- **Game**: Business logic (World, Entities, Physics).
- **Components**: Vue UI components.

## Features
- Map Creation (Voxel based)
- Characters (Steve, Alex, Villager, Iron Golem)
- Physics (Gravity, Collision)
- Interaction (Break/Place blocks)
